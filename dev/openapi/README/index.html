Installation

Place the Julia files generated under the src folder in your Julia project. Include RxInferServerOpenAPI.jl in the project code. It would include the module named RxInferServerOpenAPI.

Implement the server methods as listed below. They are also documented with the RxInferServerOpenAPI module. Launch a HTTP server with a router that has all handlers registered. A register method is provided in RxInferServerOpenAPI module for convenience.

register(
    router::HTTP.Router,        # Router to register handlers in
    impl;                       # Module that implements the server methods
    path_prefix::String="",     # Prefix to be applied to all paths
    optional_middlewares...     # Optional middlewares to be applied to all handlers
)

Optional middlewares can be one or more of:

  • init: called before the request is processed
  • pre_validation: called after the request is parsed but before validation
  • pre_invoke: called after validation but before the handler is invoked
  • post_invoke: called after the handler is invoked but before the response is sent

The order in which middlewares are invoked are: init |> read |> pre_validation |> validate |> pre_invoke |> invoke |> post_invoke
